#git 사용방법

  1. github 사이트에서 새로운 저장소를 만든다.
  2. 새로운 저장소에 들어가 clone을 한다.
  3. 터미널을 열고 git clone 주소 를 입력한다
  4. git status로 clone 이 잘 되었는지 확인한다.
  5. 작업을 진행한다.
  6. git add 로 작업한 파일을 index 위에 올린다.
  7. git commit 을 해서 commit 을 한다.
  8. git push origin 브랜치이름 을 해서 원격 저장소에 올린다.
